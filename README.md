# Somula Sudharshan Reddy — Portfolio

Built with **React + Vite + TailwindCSS + Framer Motion**

## Quick Start

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push to GitHub
2. Import repo in Vercel
3. Vercel auto-detects Vite — click Deploy

## Customize

- `src/data/profileData.js` — your name, bio, links
- `src/data/projectData.js` — add/edit projects
- `src/data/skillData.js` — update skills & levels

## Structure

```
src/
├── components/
│   ├── layout/     Navbar, Footer
│   ├── sections/   Hero, About, Skills, Projects, Education, Contact
│   └── ui/         Button, Card, Badge, SectionHeader
└── data/
    ├── profileData.js
    ├── projectData.js
    └── skillData.js
```
